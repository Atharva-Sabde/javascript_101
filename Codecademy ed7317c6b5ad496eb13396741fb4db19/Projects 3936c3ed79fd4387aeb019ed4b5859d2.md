# Projects:

- Kelvin Weather

    ```jsx
    const kelvin = 0;     //todays forecast temp

    const celcius = kelvin - 273;   // converting from kelvin to celcius.

    var fahrenheit = celcius * (9/5) +32;   //converting from celcius to fahrenheit.

    fahrenheit = Math.floor(fahrenheit);   // rounding off 
    console.log('The temperature is '+fahrenheit+' degrees Fahrenheit.');
    ```
